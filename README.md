
#  LITEX.STORRE
```
Litex.Store 是区块链支付的落地项目
```
## :earth_asia: Environment
### :wrench: 开发工具
```
01: Visual Studio code
02: Google Chrome
03: Mac
```
## :arrow_down: Setup

**Step 1:** `git clone` [repo](https://github.com/litexio/litex.store.git)

**Step 2:** `npm install`

## :arrow_forward: Run App

1. cd to the repo
2. 开发 `quasar dev`

   生产 `quasar build`
